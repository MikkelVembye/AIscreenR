# Test if the object is a `'chatgpt_tbl'` object

**\[deprecated\]**  
  
This function returns `TRUE` for `chatgpt_tbl` objects, and `FALSE` for
all other objects.

## Usage

``` r
is_chatgpt_tbl(x)
```

## Arguments

- x:

  An object

## Value

`TRUE` if the object inherits from the `chatgpt_tbl` class.
