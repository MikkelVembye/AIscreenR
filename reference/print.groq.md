# Print method for 'groq' objects

Print method for 'groq' objects

## Usage

``` r
# S3 method for class 'groq'
print(x, ...)
```

## Arguments

- x:

  A groq object from
  [`tabscreen_groq()`](https://mikkelvembye.github.io/AIscreenR/reference/tabscreen_groq.md).

- ...:

  Additional arguments passed to `print`.

## Value

Information about how to find answer data sets and pricing information.
