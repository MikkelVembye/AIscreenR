# Gemini model prices (last updated May 7, 2026) Data set containing input and output prizes for all Gemini's API models.

Gemini model prices (last updated May 7, 2026) Data set containing input
and output prizes for all Gemini's API models.

## Usage

``` r
gemini_model_prizes
```

## Format

A `data.frame` containing 4 rows/models and 3 variables/columns

|                         |             |                                       |
|-------------------------|-------------|---------------------------------------|
| **model**               | `character` | indicating the specific GPT model     |
| **price_in_per_token**  | `character` | indicating the input prize per token  |
| **price_out_per_token** | `character` | indicating the output prize per token |

## References

Gemini. *Pricing*. <https://ai.google.dev/gemini-api/docs/pricing>
